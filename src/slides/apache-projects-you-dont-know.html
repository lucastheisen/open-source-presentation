And some you may never have heard about

Apache Nifi

An easy to use, powerful, and reliable system to process and distribute data.

Nifi has an excellent web based user interface which provides visualization and management of complex data pipelines. It could be extremely useful as an ETL tool.

Apache Cordova

Mobile apps with HTML, CSS, and JS

Cordova makes mobile app developement much more approachable to your average web developer. It also provides cross platform support and compilation to native source.

Apache Calcite

The foundation for your next high-performance database

Calcite provides an industry standard SQL parser, validator and JDBC driver that you can use to SQL access to any data store. This is very promising as a way to allow SQL only tools, like Tableau, access to diverse datastores, like CSV files, MongoDB, Spark, Elasticsearch, ...

Apache Tika

A content analysis toolkit

Tika detects and extracts metadata and text from over a thousand different file types (such as PPT, XLS, and PDF). You are probably already using this one without even knowing it. Many other projects (Solr, Elasticsearch, ...) use tika to extract data during document injestion.
